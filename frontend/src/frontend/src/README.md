# Incident Timeline Visualizer

AI-powered Security Reviewer Dashboard for analyzing incident logs, generating timelines, detecting severity, and producing security recommendations.

## Features
- Login authentication
- Incident timeline generation
- Severity classification
- AI description
- AI recommendations
- AI security report
- Health check endpoint
- Rate limiting
- Input sanitization
- CSV export
- Charts dashboard
- File upload support

## Backend Run
cd backend
python app.py

## Frontend Run
cd frontend
npm run dev
